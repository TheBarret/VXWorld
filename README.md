# VXWorld

Voxel based world with simulating environmental features in python using PyGame

very early stage, slow development.

## Features
- WASD-key camera scrolling
- Mouse interaction (inspection mode)
- Painter order + frustum cull
- Precomputed draw order + sprites
- Integer projection pipeline
